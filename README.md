#  Employee Attrition & Demographics Dashboard

## Project Overview

This repository contains a **Microsoft Power BI** report designed to analyze key aspects of **employee demographics, termination patterns, and individual employee details**. The dashboard provides HR and management teams with actionable insights to understand employee turnover, identify potential areas for intervention, and support workforce planning.

##  Key Objectives

* **Analyze Termination Reasons:** Determine the most common reasons employees leave the organization.
* **Identify Demographic Trends:** Understand the distribution of employees by **Gender, Marital Status, and Race**.
* **Track Recruitment Sources:** Identify the most effective channels for sourcing new talent.
* **Provide Individual Details:** Offer a quick reference view for specific employee records, including salary and performance.

---

##  Dashboard Pages & Key Metrics

The Power BI file is organized into two main pages:

### 1. Employee Attrition & Demographics Summary (Dashboard 1)


This page provides a high-level overview of the workforce data.

| Visual | Data Analyzed | Key Insight Provided |
| :--- | :--- | :--- |
| **Termination Reason Bar Chart** | Count of Terminations by Reason | Highlights the top reasons for employee separation (**"Another position," "Unhappy,"** etc.). |
| **Gender Donut Chart** | Distribution of Employees by Gender | Shows the percentage split between **Female (F)** and **Male (M)** employees. |
| **Marital Status Donut Chart** | Distribution by Marital Status | Breaks down the workforce into **Single, Married, Divorced, and Separated** categories. |
| **Employment Status Bar Chart** | Count of Terminations vs. Active Employees | Compares the count of employees who were **Voluntarily Terminated** versus those currently employed. |
| **Race Pie Chart** | Distribution of Employees by Race | Shows the ethnic composition of the workforce. |
| **Recruitment Source Bar Chart** | Count of Hires by Source | Identifies the most successful recruitment channels (e.g., **Indeed, LinkedIn**). |

### 2. Employee Details (Dashboard 2)

This page is designed for looking up specific employee information.

| Field | Description | Example (Beak, Kimberly) |
| :--- | :--- | :--- |
| **Employee Name** | The selected employee's full name. | **Beak, Kimberly** |
| **Department** | The employee's primary department. | **Production** |
| **Gender & Marital Status** | Demographic information. | **F, Married** |
| **Date of Birth & Date of Joining** | Employee tenure and age information. | **04/17/66, 7/21/2016** |
| **Manager Name** | The employee's current direct supervisor. | **Kelley Spirea** |
| **Salary** | The employee's current annual salary. | **70K** |
| **Performance** | The employee's last recorded performance rating. | **Exceeds** |
